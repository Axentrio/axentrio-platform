# Widget appearance

The chat widget can be styled to match your brand on Pro and Enterprise.

**Tier:** custom appearance is a Pro+ feature. On Essential, the widget uses Axentrio defaults plus a small "Powered by Axentrio" badge.

**What's customisable (Pro+):**
- **Primary colour** — bubble background, send button, accent highlights
- **Position** — bottom-right (default) or bottom-left
- **Welcome message** — the text shown when the widget opens for the first time
- **Bot avatar** — upload a square image (PNG/JPG, max 500KB)
- **Hide attribution** — remove the "Powered by Axentrio" badge entirely

**Where to configure:** *AI Bot & Content → Appearance*.

**Live preview:** the appearance page shows a live preview of the widget — you can see colour changes immediately before saving.

**Mobile vs desktop:** the widget auto-adapts to screen size. On mobile it expands full-screen when opened. Colour and avatar settings apply to both.

**Reverting:** click **Reset to defaults** at the bottom of the page. The widget reverts to Axentrio's default style immediately.
