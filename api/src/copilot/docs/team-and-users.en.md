# Team and users

Multiple people can administer the same tenant — useful for handoff (operators picking up chats) and for splitting responsibilities (one person manages knowledge, another handles billing).

**Inviting a teammate:** *Settings → Team → Invite user*. Enter their email. They get an invitation link that creates their login.

**Roles:** in v1 every user is an admin — full access to settings, billing, knowledge, Leads, and inbox. Role-based permissions are on the roadmap; not yet enforced.

**Removing a teammate:** *Settings → Team → Remove user*. They lose access immediately. Any handoffs they were in the middle of go back to the open queue.

**Invite link expiry:** invite links expire after 7 days. If a teammate didn't use theirs in time, re-send from the Team page.

**Max teammates by tier:** Essential allows up to 3, Pro up to 10, Enterprise unlimited.

**Notifications per user:** each user controls their own desktop and email notifications under their personal *Profile → Notifications* settings.
