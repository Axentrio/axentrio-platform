# Enterprise plan

Enterprise is for businesses with high message volume, multiple teammates, or unique compliance needs.

**Everything in Pro, plus:**
- Highest daily LLM call cap (essentially unmetered for typical SMB volume)
- Larger file-upload limits for knowledge documents
- Multiple bots per tenant (segment customer-facing vs internal-facing knowledge)
- Multiple knowledge bases
- Priority phone support
- Custom contract terms (SLA, dedicated success contact)

**Pricing:** custom — contact sales@axentrio.com or book a call via the Help page.

**Upgrading from Pro:** Enterprise isn't self-serve — get in touch and the team will set up the upgrade with the right terms for your volume and team size.
