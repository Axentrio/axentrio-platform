HandsOff Widget Test

The chat widget should appear in the bottom-right corner.

Try these messages:

I want to book an appointment — starts the booking flow
Next Monday please — checks availability
10 AM, my name is Test User, email test@example.com — provides details
Yes — confirms the booking
Setup needed: set a tenant API key in localStorage before the widget can connect. Open DevTools console and run:

localStorage.setItem('cb_test_api_key', 'YOUR_TENANT_API_KEY')
localStorage.setItem('cb_test_api_url', 'http://localhost:4081') (optional — defaults to Railway prod)
localStorage.setItem('cb_test_widget_src', '/chatbot-platform/api/public/widget.js') (optional — local file vs API-served)

Then reload the page.
